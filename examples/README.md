## Share Your Own Example

Did you run your CV through the screener — especially one that got **ACCEPTED FOR INTERVIEW**, or one where the verdict matched what actually happened with the real company? We'd love to publish it.

Open an issue using the **"Share a screening example"** template, or submit a PR adding a file to this folder with the same three-part structure (job description, CV, full output).

**Before sharing, anonymize everything:** remove your name, contact details, and any identifying information. Replace specific employers with descriptions if needed ("top-3 Italian bank", "Series B SaaS startup"). We will not merge examples containing personal data.
